# 🏧 ATM Simulation System using Python

A console-based ATM Simulation project built using Python to replicate real-world banking operations. This project demonstrates strong understanding of core programming concepts and logical flow control.

---

## 📌 About the Project

The ATM Simulation System allows users to perform basic banking operations in a menu-driven interface. It continuously runs until the user chooses to exit, ensuring a smooth and interactive experience.

---

## 🚀 Features Implemented

* ✅ Balance Inquiry
* ✅ Deposit Money
* ✅ Withdraw Money
* ✅ Input Validation
* ✅ Error Handling
* ✅ Loop-based continuous execution
* ✅ Function-based modular design

---

## 🛠️ Concepts Used

* Python Functions
* While Loops
* Conditional Statements (if-else)
* Exception Handling
* Logical Flow Control
* State Management
